# React-projects
collection of beginners and intermediate-level React projects


## Tech Stack
- React
- TypeScript
- Tailwind CSS
- Vite

## How to use
1. Clone the repo
``` bash
git clone https://your-repo-url.git
```

2. Go to the project you want to run

3. Install dependencies
``` bash
npm install
```

4. Start the dev server
``` bash
npm run dev
